# watch-tx
在 terminal 用 shell script 持續觀察台指期報價，方便看盤

<br>

## 支援 OS
- 目前只支援 `MacOS` 和 `Linux`

<br>

## 需要安裝的套件

- `jq`

### Linux

```bash
sudo apt-get install jq
```

<br>

### MacOS

```bash
brew install jq
```

<br>

## 使用方法

```bash
curl -s https://raw.githubusercontent.com/ktlast/watch-tx/master/watch_tx.sh | bash
```
